pitch版本：CSS2　 兼容性：无　继承性：有
语法：
pitch : x-low | low | medium | high | x-high | number
取值：
number :数字。指定赫兹（HZ）值。其他参数取决于 voice-family 属性指定的声音种类。请参阅 频率单位
说明：
设置或检索音高。
对应的脚本特性为 pitch 。
示例：
strong { pitch: x-high }
strong { pitch: 75hz }
