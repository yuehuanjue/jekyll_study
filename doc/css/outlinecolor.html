outline-color版本：CSS2　 兼容性：无　继承性：无
语法：
outline-color : color | invert
取值：
color :指定颜色。请参阅 颜色单位 和 附录：颜色表
invert :默认值。使用背景色的反色
说明：
设置或检索对象外的线条轮廓( outline )的颜色。参见属性。
对应的脚本特性为 outlineColor 。
示例：
img { outline-color: red }
p { outline-color: #E9E9E9 }
