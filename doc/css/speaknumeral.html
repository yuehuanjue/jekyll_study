speak-numeral版本：CSS2　 兼容性：无　继承性：有
语法：
speak-numeral : continuous | digits
取值：
continuous :默认值。根据默认语言对数字的词语描述阅读
digits :逐位阅读数字
说明：
设置或检索数字如何发音。
对应的脚本特性为 speakNumeral 。
示例：
.telephone { speak-numeral: digits }
