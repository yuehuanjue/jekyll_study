outline版本：CSS2　 兼容性：无　继承性：无
语法：
outline : outline-color || outline-style || outline-width
取值：
该属性是复合属性。请参阅各参数对应的属性。
说明：
设置或检索对象外的线条轮廓。外轮廓( outline )画在边框( border )外面，并且不一定是矩形。
对应的脚本特性为 outline 。
示例：
img { outline: red }
p { outline: double 5px }
button { outline: #E9E9E9 double thin }
