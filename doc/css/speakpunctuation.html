speak-punctuation版本：CSS2　 兼容性：无　继承性：有
语法：
speak-punctuation : none | code
取值：
none :默认值。取消标点符号发音。替代为停顿
code :标点符号被发音
说明：
设置或检索标点符号如何发音。
对应的脚本特性为 speakPunctuation 。
示例：
.telephone { speak-punctuation: code; speak-numeral: digits }
