outline-width版本：CSS2　 兼容性：无　继承性：无
语法：
outline-width : medium | thin | thick | length
取值：
medium :默认值。默认宽度
thin :小于默认宽度
thick :大于默认宽度
length :由浮点数字和单位标识符组成的长度值。不可为负值。请参阅 长度单位
说明：
设置或检索对象外的线条轮廓( outline )的宽度。
对应的脚本特性为 outlineWidth 。
示例：
img { outline-color: orange; outline-style: solid ; outline-width: medium ; }
