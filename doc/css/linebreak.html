line-break版本：IE5+专有属性　继承性：有
语法：
line-break : normal | strict
取值：
normal : 默认值。应用日文文本的默认换行规则
strict : 强制日文文本换行规则的严谨性
说明：
设置或检索用于日文文本的换行规则。
此属性对于 currentStyle 对象而言是只读的。对于其他对象而言是可读写的。
对应的脚本特性为 lineBreak 。
示例：
div { line-break : strict; }
