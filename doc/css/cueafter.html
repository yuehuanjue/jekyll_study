cue-after版本：CSS2　 兼容性：无　继承性：无
语法：
cue-after : none | url ( url )
取值：
none :默认值。无声音播放
url ( url ) :使用绝对或相对 url 地址指定音乐文件
说明：
设置在对象后播放的声音。
对应的脚本特性为 cueAfter 。
示例：
div { cue-after: url(orchestralsneeze.wav) ; }
