:hover版本：CSS1/CSS2　 兼容性：IE4+ NS4+
语法：
Selector : hover { sRules }
说明：
设置对象在其鼠标悬停时的样式。
在CSS1中此伪类仅可用于 a 对象。对于无 href 属性(特性)的 a 对象，此伪类不发生作用。
在CSS2中此伪类可以应用于任何对象。
示例：
a:hover { font-size: 14pt; text-decoration: underline; color: blue; }
a:hover span{ color:red; }
