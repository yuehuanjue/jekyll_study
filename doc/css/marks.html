marks版本：CSS2　 兼容性：无　继承性：无
语法：
marks : none | crop || cross
取值：
none :无页标记出现
crop :在页面将会被裁切处指定切割标记
lcross :指定交叉线标记用来供页面对齐
说明：
印刷业的印刷文档通常在页面内容区域外带有一些标记，用来校正和调整所有的页面。此属性用来设置或检索什么样的标志是应该在页容器外边被给予。
对应的脚本特性为 marks 。
示例：
body { marks: crop cross }
