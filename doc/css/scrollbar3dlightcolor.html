scrollbar-3dlight-color版本：IE5.5+专有属性　继承性：有
语法：
scrollbar-3dlight-color : color
取值：
color :指定颜色。请参阅 颜色单位 和 附录：颜色表
说明：
设置或检索滚动条亮边框颜色。
请参阅 overflow 属性 。
对应的脚本特性为 scrollbar3dLightColor 。
示例：
div {scrollbar-3dlight-color :threedhighlight; }
