speak版本：CSS2　 兼容性：无　继承性：有
语法：
speak : normal | none | spell-out
取值：
normal :默认值。使用当前语言的正常发音尺度
none :取消发音。不同于 volume 的 silent 值，未发出的声音不占用时间。这个值不会被子对象继承
spell-out :一次一字的拼写内容
说明：
设置或检索声音是否给出。
对应的脚本特性为 speak 。
示例：
acronym { speak: spell-out }
