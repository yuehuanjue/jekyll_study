ruby-position版本：IE5+专有属性　继承性：无
语法：
ruby-position : above | inline
取值：
above :默认值。在基本文本上方定位 ruby 文本
inline : ruby 文本与基本文本内联
说明：
设置或检索通过 rt 对象指定的注释文本或发音指南(参考 ruby 对象)的位置。
此属性对于 currentStyle 对象而言是只读的。对于其他对象而言是可读写的。
对应的脚本特性为 rubyPosition 。
示例：
ruby { ruby-position: above; }
