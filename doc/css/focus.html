:focus版本：CSS2　 兼容性：无
语法：
Selector : focus { sRules }
说明：
设置对象在成为输入焦点(该对象的 onfocus 事件发生)时的样式。
示例：
a:focus { font-size: 14pt; text-decoration: underline; color: blue; }
a:focus img { border: thin solid green }
