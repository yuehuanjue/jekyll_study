ruby-overhang版本：IE5+专有属性　继承性：无
语法：
ruby-overhang : auto | whitespace | none
取值：
auto :默认值。 ruby 文本突出相邻基本文本的任何其他文本
whitespace : ruby 文本只突出空白区域的字符
none : ruby 文本只突出相邻基本文本的任何其他文本
说明：
设置或检索通过 rt 对象指定的注释文本或发音指南(参考 ruby 对象)的位置。
此属性对于 currentStyle 对象而言是只读的。对于其他对象而言是可读写的。
对应的脚本特性为 rubyOverhang 。
示例：
ruby { ruby-overhang: auto; }
