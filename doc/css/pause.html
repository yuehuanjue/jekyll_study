pause版本：CSS2　 兼容性：无　继承性：无
语法：
pause : pause-before || pause-after
取值：
该属性是复合属性。请参阅各参数对应的属性。
说明：
设置对象前后的声音暂停。
假如只给出了一个值，它将用于 pause-before 属性和 pause-after 属性。假如给出了两个值，则第一个用于 pause-before 属性，第二个用于 pause-after 属性。
对应的脚本特性为 pause 。
示例：
div { pause: 2s 3s }
div { pause: 2s; }
