layer-background-color版本：NS4+专有属性　继承性：无
语法：
layer-background-color : transparent | color
取值：
transparent :默认值。背景色透明
color :指定颜色。请参阅 颜色单位 和 附录：颜色表
说明：
设置或检索对象整个区域的背景颜色。
示例：
div { layer-background-color: transparent; }
