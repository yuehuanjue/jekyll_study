返回值:Array<Element(s)>:disabled

V1.0概述

匹配所有不可用元素

示例

描述:

查找所有不可用的input元素

HTML 代码:
<form>
  <input name="email" disabled="disabled" />
  <input name="id" />
</form>
jQuery 代码:
$("input:disabled")
结果:
[ <input name="email" disabled="disabled" /> ]
