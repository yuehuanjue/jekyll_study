V1.0概述

匹配所有重置按钮

示例

描述:

查找所有重置按钮

HTML 代码:
<form>
  <input type="text" />
  <input type="checkbox" />
  <input type="radio" />
  <input type="image" />
  <input type="file" />
  <input type="submit" />
  <input type="reset" />
  <input type="password" />
  <input type="button" />
  <select><option/></select>
  <textarea></textarea>
  <button></button>
</form>
jQuery 代码:
$(":reset")
结果:
[ <input type="reset" /> ]
