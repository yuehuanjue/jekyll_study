返回值:jQuerynextAll([expr])

概述

查找当前元素之后所有的同辈元素。

可以用表达式过滤

参数

expr StringV1.2

用来过滤的表达式

示例

描述:

给第一个div之后的所有元素加个类

HTML 代码:
<div></div><div></div><div></div><div></div>
jQuery 代码:
$("div:first").nextAll().addClass("after");
结果:
[ <div class="after"></div>, <div class="after"></div>, <div class="after"></div> ]
