返回值:Array<Element(s)>:enabled

V1.0概述

匹配所有可用元素

示例

描述:

查找所有可用的input元素

HTML 代码:
<form>
  <input name="email" disabled="disabled" />
  <input name="id" />
</form>
jQuery 代码:
$("input:enabled")
结果:
[ <input name="id" /> ]
