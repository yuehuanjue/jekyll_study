返回值:jQueryempty()

V1.0概述

删除匹配的元素集合中所有的子节点。

示例

描述:

把所有段落的子元素（包括文本节点）删除

HTML 代码:
<p>Hello, <span>Person</span> <a href="#">and person</a></p>
jQuery 代码:
$("p").empty();
结果:
<p></p>
