返回值:IntegerinnerWidth()

V1.2.6概述

获取第一个匹配元素内部区域宽度（包括补白、不包括边框）。

此方法对可见和隐藏元素均有效。

示例

描述:

获取第一段落内部区域宽度。

HTML 代码:
<p>Hello</p><p>2nd Paragraph</p>
jQuery 代码:
var p = $("p:first");
$("p:last").text( "innerWidth:" + p.innerWidth() );
结果:
<p>Hello</p><p>innerWidth: 40</p>
