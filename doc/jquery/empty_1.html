返回值:Array<Element(s)>:empty

V1.0概述

匹配所有不包含子元素或者文本的空元素

示例

描述:

查找所有不包含子元素或者文本的空元素

HTML 代码:
<table>
  <tr><td>Value 1</td><td></td></tr>
  <tr><td>Value 2</td><td></td></tr>
</table>
jQuery 代码:
$("td:empty")
结果:
[ <td></td>, <td></td> ]
