返回值:StringjQuery.trim(str)

概述

去掉字符串起始和结尾的空格。

参数

strStringV1.0

需要处理的字符串

示例

描述:

去掉字符串起始和结尾的空格。

jQuery 代码:
$.trim("  hello, how are you?  ");
结果:
"hello, how are you?"
